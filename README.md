# 🛡️ GovScan – Government Security Monitoring System

## 🚨 Problem
Government websites often lack continuous automated monitoring for security vulnerabilities.

## 💡 Solution
GovScan is an automated system that scans, analyzes, and monitors website security with real-time risk assessment and prioritization.

## ⚙️ Features
- 🔍 Website vulnerability scanning (SSL, headers, ports, SQLi)
- 📊 Security score (0–100)
- 🚨 Alert system for high risk
- 📈 Trend graph (historical tracking)
- 🧠 Priority fix engine (what to fix first)
- 🌐 Tamil language support
- 🤖 Agent-based continuous monitoring

## 🧱 Tech Stack
- Python (Flask)
- SQLite
- Chart.js
- HTML/CSS/JS
